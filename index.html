Rules

  1. Each player starts with 5 dice.
  2. Roll all dice still in play.
  3. 3's count as 0 points.
  4. The player selects/removes their own 3's. If there are no 3's, select one or more dice to remove.
  5. Keep rolling until no dice remain.
  6. Lowest score wins.
